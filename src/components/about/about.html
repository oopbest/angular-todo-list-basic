Angular TodoList Basic

โปรเจคนี้สร้างขึ้นเพื่อเรียนรู้ AngularJS 1.5.11 ตั้งแต่พื้นฐาน

🛠️ เทคโนโลยีที่ใช้

  • AngularJS 1.5.11 — JavaScript Framework สำหรับสร้าง SPA
  • UI-Router — ระบบ Routing แบบ State-based
  • Webpack — Module Bundler รวมไฟล์ทั้งหมดเข้าด้วยกัน
  • LESS — CSS Preprocessor เพิ่มพลังให้ CSS
  • html-loader — โหลด HTML templates เป็น string

📚 สิ่งที่ได้เรียนรู้

  • การใช้ ng-repeat, ng-model, ng-click, ng-class, ng-if
  • การสร้าง Controller ด้วย $scope
  • การใช้ Filter ใน AngularJS
  • การตั้งค่า Routing ด้วย UI-Router
  • การใช้ Webpack Loaders (less-loader, html-loader)
